# ATR Engine Marketplace Index

Central index for the Advanced Tape Restorer engine marketplace.

This repository hosts `engines-index.json`, the catalog of installable add-on engines
for ATR v6.0+. The ATR Add-ons page fetches this file to display available engines.

## Available Engines

| Engine | Version | License | Description |
|--------|---------|---------|-------------|
| LTX Generative Repair | 1.1.0 | Professional | AI-powered generative video repair using LTX-Video diffusion models |

## For Engine Authors

Add your engine to the marketplace by submitting a PR with an entry in `engines-index.json`.

See the [Plugin Marketplace Design Doc](https://github.com/idocinthebox/ATR-LTX-Generative-Repair/blob/main/DESIGN.md) for the `.atrengine` package format.
